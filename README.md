📊 Customer Churn Prediction Dashboard

🚀 Overview

This project is an end-to-end machine learning solution to predict whether a customer is likely to churn based on their behavior and transaction data. It includes data analysis, model building, and an interactive web application.

---

🎯 Problem Statement

Customer churn is a major challenge in e-commerce businesses. This project aims to:

- Identify customers at risk of leaving
- Analyze key factors influencing churn
- Provide actionable insights for retention strategies

---

🧠 Features

- 📌 Data cleaning and preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning model (Random Forest)
- 📈 Churn prediction with probability score
- 🌐 Interactive web app using Streamlit
- 💡 Business insights based on customer behavior

---

🛠 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit

▶️ How to Run the Project

1. Clone the repository

git clone https://github.com/your-username/customer_churn_prediction.git
cd customer_churn_project

2. Install dependencies

pip install -r requirements.txt

3. Train the model

python src/train_model.py

4. Run the application

streamlit run app.py

📊 Model Performance

- Accuracy: ~93%
- Model: Random Forest Classifier

---

💡 Key Insights

- Customers with low satisfaction scores are more likely to churn
- Higher days since last order increases churn probability
- Customers with fewer interactions tend to leave
